# 🤖 クロードコード（Claude Code）使い方ガイド

> OSHINAVI作業のおとも、Claude Code の早見表よ。
> 困ったらこのファイルを見返してね。

---

## 📌 これは何？

**Claude Code** ＝ ターミナル（黒い画面）で動くAIアシスタント（＝あたし）。
日本語で「これやって」と頼むと、ファイルを読んで・調べて・書き換えてくれる相棒です。

- **VS Code**：自分の手でコードを書く「道具・作業場」
- **Claude Code**：頼むと代わりに作業してくれる「働き手」

---

## ⌨️ 基本のキー操作（まずこれだけ覚えればOK）

| キー | できること |
|---|---|
| `Enter` | メッセージを送信 |
| `Esc` | 作業中のClaudeを**止める**（途中で中断したいとき） |
| `Esc` を2回 | ひとつ前の自分のメッセージに**戻って編集**（巻き戻し） |
| `↑` `↓` | 過去に打った入力を呼び出す |
| `Shift + Tab` | モード切替（通常 ⇄ 自動承認 ⇄ プランモード） |
| `Ctrl + C` | 入力の取り消し（2回で終了） |
| `Ctrl + L` | 画面をきれいに消す（会話は消えない） |

---

## ✨ 特殊な入力（行の先頭で使う記号）

| 記号 | 意味 | 例 |
|---|---|---|
| `/` | **コマンド**を呼び出す（下の一覧参照） | `/help` |
| `!` | **シェルコマンドを実行**（ログインなど自分で打ちたい時） | `! git status` |
| `#` | いま伝えたことを**メモリに記憶**させる | `# pushは1日2回まで` |
| `@` | **ファイルを指定**して読ませる | `@plan.md を確認して` |

---

## 🛠️ よく使うスラッシュコマンド（`/` から始まる機能）

> 💡 入力欄で `/` を打つと、**使える全コマンドの一覧がその場で出る**わ。
> これが一番確実。下は特によく使うものだけ抜粋。

### 基本
| コマンド | できること |
|---|---|
| `/help` | ヘルプ・使い方を表示 |
| `/clear` | 会話をまっさらにリセット（話が長くなったら） |
| `/compact` | 会話を要約して軽くする（リセットせず続けたい時） |
| `/config` | 設定画面（テーマ・モデルなど） |
| `/model` | 使うAIモデルを切り替え |
| `/fast` | 高速モードのオン・オフ（Opusのまま速くなる） |
| `/cost` | これまでの利用コストを表示 |
| `/resume` | 前の会話の続きから再開 |
| `/memory` | 記憶（メモリ）の内容を見る・編集 |

### OSHINAVI作業で役立つもの
| コマンド | できること |
|---|---|
| `/loop` | 同じ作業を一定間隔でくり返す（例：5分ごとにチェック） |
| `/schedule` | 決まった時刻に自動でタスクを動かす（朝のルーチンなど） |
| `/deep-research` | ネットを横断して深く調べてレポートにする |
| `/run` | サイト（アプリ）を実際に立ち上げて動作確認 |
| `/code-review` | コードの変更をレビューして問題を指摘 |

## 🌅 OSHINAVIでのあたしの使い方（参考）

毎朝こんな風に頼めば、ルーチンが回るわ：

- 「**おはよう、朝のルーチンお願い**」
  → plan.md確認 → 期限切れチェック → 新着取得…の流れ
- 「**期限切れ消していい？**」と聞かれたら、自分でサイト検索して確認 → 「削除して」
- 「**コミットして**」「**プッシュして**」は、あなたのOKが出てから実行（無断ではやらない）

---

## 🆘 困ったとき

| こんな時 | こうする |
|---|---|
| Claudeが変な方向に進んでる | `Esc` で止めて、やり直しを指示 |
| さっきの指示を間違えた | `Esc` を2回押して書き直す |
| 話が長くなって重い・混乱してる | `/compact` で要約、ダメなら `/clear` |
| どんなコマンドがあるか忘れた | 入力欄で `/` を打つ（全部出る） |
| コマンドを自分で実行したい | 行頭に `!` をつける（例：`! git log`） |

---

## 📝 覚えておくと安心

- **会話は記憶される**：閉じても `/resume` で続きから再開できる
- **メモリ機能**：`#` で伝えたルールは次回以降も覚えてる（OSHINAVIのルールも記憶済み）
- **無断で危ないことはしない**：削除・push などは必ず確認してから
